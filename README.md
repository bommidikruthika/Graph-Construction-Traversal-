# Graph Construction and Traversal Visualizer

## 📌 Project Description
This project is a Graph Construction and Traversal Visualizer developed using Python.

It allows users to:
- Add nodes
- Add edges
- Perform BFS Traversal
- Perform DFS Traversal
- Find shortest path using Dijkstra Algorithm
- Visualize graphs interactively

---

## 🚀 Technologies Used
- Python
- CustomTkinter
- NetworkX
- Matplotlib

---

## 📂 Features
✔ Graph Construction  
✔ BFS Traversal  
✔ DFS Traversal  
✔ Dijkstra Algorithm  
✔ Graph Visualization  
✔ GUI Interface  

---

## ▶️ How to Run

Install required libraries:

```bash
pip install customtkinter networkx matplotlib
```

Run project:

```bash
python gui.py
```
